# Mac setup

Run the ```setup.sh``` script to install and configure a new mac. The script will install the following:
- Homebrew
- Brew casks
- Brew formulae
- NVM
- Node LTS
- Prettier
- ESLint
- Oh my zsh
- Starship

It will also:
- Configure my git user
- Set node version to LTS
- Create a dev dir in ```~```
